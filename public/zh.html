Neko's Chat Matrix 服务器

介绍

Neko's Chat 是一个基于 Matrix 协议的开源即时通讯平台。Matrix 是一种去中心化的通信协议，允许用户跨服务器连接并互通，确保数据隐私和安全。无论你是想与朋友聊天，还是创建一个社区讨论组，Neko's Chat 都是理想的选择。

推荐的应用程序

为了更好地使用 Neko's Chat Matrix 服务器，我们推荐以下应用程序：

  • Element: 功能强大的 Matrix 客户端，适用于 iOS、Android、Windows、macOS、Linux 和 Web。Element 提供了最全面的账户管理选项，适合需要高级控制的用户。下载地址：Element 下载 | Element Web 客户端 | Element Android（无需 Google Play）
  • Element X: 轻量级的 Matrix 客户端，适用于 iOS 和 Android。Element X 设计简洁，易于使用，适合快速上手的用户。下载地址：Element X 下载 | Element X Android（无需 Google Play）
  • FluffyChat: 简洁轻便的 Matrix 客户端，适用于 iOS、Android、Web 和 Linux。FluffyChat 支持多用户账户切换，适合管理多个账户的用户。下载地址：FluffyChat 下载 | FluffyChat Web 客户端

如何注册

如果你想加入 Neko's Chat，请发送邮件至 hi@nekos.chat 申请注册。请在邮件中提供你想要的用户名，用户名的格式应为 @xxx:nekos.chat。

一旦我们收到你的邮件，将发送账户密码给你。请在收到密码后回复邮件确认，然后你就可以使用你的账户登录。

需要帮助？

如果你在使用过程中遇到问题或有任何疑问，例如需要重置密码，请随时通过邮件联系我们。你可以发送邮件到 hi@nekos.chat，我们会尽快帮助你解决问题。

报告滥用

如果你遇到任何滥用行为，或者需要报告服务器的滥用问题，请通过电子邮件联系我们 hi@nekos.chat，联系服务器管理员 @test111:nekos.chat，或在专门的滥用反馈群组 #abuse:nekos.chat 进行报告。我们会及时处理你的报告。
