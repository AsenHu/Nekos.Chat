Welcome to Neko's Chat Matrix Server

Introduction

Neko's Chat is an open-source instant messaging platform based on the Matrix protocol. Matrix is a decentralized communication protocol that allows users to connect across different servers, ensuring data privacy and security. Whether you want to chat with friends or create a community discussion group, Neko's Chat is your ideal choice.

Recommended Applications

To make the best use of the Neko's Chat Matrix server, we recommend the following applications:

  • Element: A powerful Matrix client available for iOS, Android, Windows, macOS, Linux, and Web. Element provides the most detailed and comprehensive account management settings, making it perfect for users who need advanced control. Download links: Element Download | Element Web Client | Element Android (No Google Play)
  • Element X: A lightweight Matrix client available for iOS and Android. Element X has a simple design and is easy to use, making it especially suitable for users who want to get started quickly. Download links: Element X Download | Element X Android (No Google Play)
  • FluffyChat: A sleek and lightweight Matrix client available for iOS, Android, Web, and Linux. FluffyChat supports multiple user switching, making it ideal for users who manage multiple accounts. Download links: FluffyChat Download | FluffyChat Web Client

How to Register

If you would like to join Neko's Chat, please send an email to hi@nekos.chat to request registration. Please include your desired username in the email, and the username format should be @xxx:nekos.chat.

Once we receive your email, we will send you an account password. Please reply to the password email to confirm receipt, after which your account will be ready for login.

Need Help?

If you encounter any issues or have any questions, such as needing to reset your password, please feel free to contact us via email. You can send an email to hi@nekos.chat, and we will assist you as soon as possible.

Report Abuse

If you encounter any abusive behavior or need to report any issues related to server misuse, please contact us via email at hi@nekos.chat, or report the issue in our dedicated abuse group #abuse:nekos.chat. We take abuse reports seriously and will respond promptly.

Thank you for your support and interest in Neko's Chat!
